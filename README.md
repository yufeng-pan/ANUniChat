# ANUniChat

## Description
ANUniChat is a chat application designed for real-time communication. This project showcases my skills in full-stack development, featuring a backend built with Node.js and Express, and a frontend developed using React. The application offers a seamless and interactive chat experience, demonstrating my ability to create comprehensive and user-friendly web applications.

## Live Demo
Experience ANUniChat live at: [https://anunichat.onrender.com](https://anunichat.onrender.com)

## Technology Stack
- **Backend**: Node.js, Express
- **Frontend**: React
- **Database**: MongoDB
- **Other Technologies**: Socket.IO for real-time communication, JWT for authentication

## Features
- Real-time messaging capabilities.
- Group Chat Creation and Management.
- User authentication and profile management.
- User Search Functionality.
- Responsive and intuitive user interface.
